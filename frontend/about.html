About the Project

📌 Problem Statement

E-commerce platforms often struggle with inventory management due to unpredictable and fluctuating customer demand. Poor demand estimation leads to overstocking or frequent stock shortages, resulting in financial losses and reduced customer satisfaction. Traditional manual forecasting methods are insufficient to handle large-scale and dynamic sales data.

💡 Proposed Solution

DemandSense AI is a machine learning-based demand forecasting system that predicts future product demand using historical sales patterns and contextual inputs such as price and date. The system provides product-level predictions, category-wise analytics, and actionable inventory recommendations through an interactive web interface.

⚙️ System Architecture

  • User interacts with the web-based frontend
  • Frontend sends input data to Flask backend via REST API
  • Machine learning model processes inputs and predicts demand
  • Results are visualized using charts and dashboards

🛠 Technology Stack

  • Frontend: HTML, CSS, JavaScript, Chart.js
  • Backend: Python, Flask
  • Machine Learning: Random Forest Regressor (scikit-learn)
  • Tools: VS Code, GitHub

🎯 Applications & Impact

  • Improves inventory planning and stock optimization
  • Reduces losses due to overstocking and understocking
  • Supports data-driven decision-making in retail
  • Scalable for real-world e-commerce platforms
